# EasternLabs Landing Page

Landing page for EasternLabs — A Studio for Digital Craft.

## Development

### Setup

```bash
npm install
```

### Run Development Server

```bash
npm run dev
# or
npm start
```

The development server will start on `http://localhost:3000` with live reload enabled.

## Project Structure

```
.
├── index.html          # Main HTML file
├── css/
│   └── main.css       # Custom styles
├── js/
│   └── main.js        # JavaScript functionality
├── CNAME              # GitHub Pages custom domain
├── package.json        # Node.js dependencies
└── README.md          # This file
```

## Technologies

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework (via CDN)
- **Vanilla JavaScript** - No framework dependencies
- **Google Fonts** - VT323 & Roboto Mono

## Deployment

This site is deployed via GitHub Pages. The `CNAME` file configures the custom domain `easternlabs.app`.
